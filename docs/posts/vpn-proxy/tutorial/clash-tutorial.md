# Clash使用教程：从入门到精通

Clash是一款功能强大的代理工具，本文将详细介绍其使用方法。

## 下载安装

### Windows
1. 前往官网下载
2. 解压到指定目录
3. 运行程序

### macOS
1. 下载dmg文件
2. 拖入应用程序文件夹

## 配置说明

### 导入配置
- 点击配置按钮
- 选择导入URL
- 输入订阅地址

### 规则设置
- 全局模式
- 规则模式
- 直连模式

## 常见问题

Q: 为什么无法连接？
A: 检查订阅地址是否正确。
