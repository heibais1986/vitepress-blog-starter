# 翻墙入门指南

这篇文章将手把手教你如何开始科学上网。

## 🎯 你将学到

- 什么是VPN和代理
- 如何选择合适的工具
- 如何配置和使用
- 常见问题解决

## 📖 第一步：了解基础概念

### 什么是科学上网？

科学上网是指通过技术手段访问被限制的网络资源。

### 常用工具

1. **V2Ray** - 功能强大，配置灵活
2. **Clash** - 界面友好，适合新手
3. **Shadowsocks** - 轻量级，速度快

## 🚀 第二步：选择客户端

### Windows系统
- V2RayN
- Clash for Windows

### macOS系统
- ClashX
- V2RayX

### 手机端
- iOS: Shadowrocket, Quantumult
- Android: V2RayNG, Clash

## 💡 第三步：获取节点

1. 免费节点（不稳定）
2. 付费机场（推荐）
3. 自建节点（高级）

## ✅ 验证是否成功

打开浏览器访问：
- Google.com
- YouTube.com
- Twitter.com

如果能正常访问，说明配置成功！
