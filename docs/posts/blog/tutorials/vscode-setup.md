# VS Code最佳配置指南2024

打造高效的VS Code开发环境。

## 必装插件

### 通用插件
- Chinese Language Pack
- Material Icon Theme
- GitLens

### 前端开发
- ESLint
- Prettier
- Auto Rename Tag

### Python开发
- Python
- Pylance

## 配置优化

### settings.json
```json
{
  "editor.fontSize": 14,
  "editor.tabSize": 2,
  "files.autoSave": "onFocusChange"
}
```

## 快捷键

- Ctrl+P：快速打开文件
- Ctrl+Shift+P：命令面板
- Alt+↑/↓：移动行
