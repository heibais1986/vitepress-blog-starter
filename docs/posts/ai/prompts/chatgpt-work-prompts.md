# ChatGPT工作提示词合集

精选适合工作场景的ChatGPT提示词。

## 邮件撰写

```
请帮我写一封专业的商务邮件，主题是...
```

## 报告总结

```
请帮我总结以下内容的核心要点：
[粘贴内容]
```

## 会议记录

```
请帮我整理这次会议的要点和待办事项：
[粘贴会议内容]
```

## PPT大纲

```
请帮我制作一个关于[主题]的PPT大纲，
包含5-7个部分。
```
