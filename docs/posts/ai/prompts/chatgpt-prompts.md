# ChatGPT提示词大全

精选100+个ChatGPT提示词，复制即用。

## 📝 写作类

### 公众号爆款标题生成器

```
你是一个资深的新媒体运营专家，擅长撰写吸引眼球的标题。

请根据以下文章内容，生成10个公众号标题，要求：
1. 能引发好奇心
2. 包含数字或关键词
3. 符合微信生态
4. 长度控制在20字以内

文章内容：[在这里粘贴你的文章内容]
```

### 小红书文案生成

```
你是小红书爆款文案专家，请帮我写一篇小红书笔记。

主题：[你的主题]
要求：
- 开头要吸引人
- 多用emoji表情
- 分段要清晰
- 结尾引导互动
- 字数500字左右
```

## 💼 工作类

### 周报生成器

```
你是一个专业的职场助手。请根据我本周的工作内容，帮我生成一份专业的周报。

本周工作：
1. [工作内容1]
2. [工作内容2]
3. [工作内容3]

格式要求：
- 本周工作总结
- 重点成果
- 遇到的问题
- 下周计划
```

### PPT大纲生成

```
请帮我生成一个PPT大纲。

主题：[演讲主题]
时长：[演讲时长]
听众：[听众类型]

要求：
- 逻辑清晰
- 包含5-7个核心要点
- 每个要点有3-4个子内容
- 开头有引言，结尾有总结
```

## 💻 编程类

### 代码解释器

```
请详细解释以下代码的功能，包括：
1. 整体功能说明
2. 逐行代码解释
3. 可能的优化建议

代码：
[粘贴你的代码]
```

### Bug调试助手

```
我的代码出现了错误，请帮我分析原因并提供解决方案。

错误信息：[错误信息]
相关代码：[代码片段]
运行环境：[环境信息]
```

## 🎓 学习类

### 概念解释

```
请用简单易懂的语言解释以下概念，要求：
1. 先给一句话定义
2. 举一个生活中的例子
3. 说明它的实际应用
4. 适合小学生理解

概念：[概念名称]
```

### 知识总结

```
请帮我总结以下内容，生成思维导图大纲：

内容：[要总结的内容]

要求：
- 提炼核心要点
- 层级清晰
- 包含关键词
```

::: tip 使用技巧
1. 根据自己的需求修改方括号[]内的内容
2. 可以让ChatGPT按照你的风格继续优化
3. 保存常用的提示词，提高效率
:::
